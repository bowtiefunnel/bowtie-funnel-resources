/website-visitor-deanon

Anonymous traffic is your warmest, most-wasted audience — people researching you right now, and you never learn who.

This skill puts names on them. It resolves anonymous visitors to real contacts and companies, qualifies them, and routes them — as net-new leads, or as an activity signal on clients you already have.

MIT Claude Code skill Awareness Retention runs on Growth MCP

The repurpose — raw tool → Bowtie skill

before — raw MCP
  • Two bare tools: deanonymize_contacts → get_results
  • Output is a private artifact_id inside Verbiflow
  • No positioning, no deliverable, no reuse
  • Invisible to the client
→
after — Bowtie Funnel skill
  • Rebranded + placed at Awareness & Retention on the bowtie
  • Wrapped in a play: de-anon → enrich → qualify → route
  • Output published to a branded labs URL, not a CSV
  • Reused by aim: new traffic = leads; client domains = account signal
anonymous traffic→de-anon→enrich→qualify→labs.bowtiefunnel.com/<client>/

How it runs

  1. growth_session_init — auth + workspace.
  2. growth_deanonymize_contacts — resolve anonymous traffic to contacts/companies.
  3. growth_get_deanonymize_contacts_results — page the rows (returns an artifact_id).
  4. growth_enrich_emails / growth_fetch_person_profiles — fill emails, titles, attributes.
  5. growth_qualify_rows — score against the ICP; keep A/B fit (caps at 500 rows/call).
  6. Route: unknown companies → sequence or sales (Awareness); existing-customer domains → at-risk / activity signal (Retention).

/plugin marketplace add bowtiefunnel/bowtie-funnel-Labs
/plugin install bowtie-gtm@bowtie-funnel

Bundles this skill (and the rest) plus the Growth MCP server, wired up.

Option B — just this skill

# 1) add the Growth MCP once (the engine the skill calls)
claude mcp add --transport stdio growth -- npx -y growth-mcp

# 2) copy this one skill into your Claude Code skills dir
cp -r plugins/bowtie-gtm/skills/website-visitor-deanon ~/.claude/skills/

Then invoke it, or just say what you want:

/website-visitor-deanon

# or, in plain language —
# "who visited our site this week?"
# "de-anonymize last week's traffic and score it against our ICP"
