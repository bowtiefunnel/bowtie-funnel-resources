/buying-intent-miner

The warmest prospects are already raising their hands — commenting, reacting, posting about your problem.

Buying-Intent Miner. It mines LinkedIn posts, comments, and reactions to surface people showing intent, before a single cold email.

MIT Claude Code skill Awareness Expansion runs on Growth MCP

The repurpose — raw tool → Bowtie skill

before — raw MCP
  • linkedin_search_posts + post_comments + post_reactions
  • Raw engagement data
  • No enrichment or routing
→
after — Bowtie Funnel skill
  • Rebranded + placed at Awareness & Expansion
  • Wrapped in a play: pick a topic/post → pull engagers → enrich + sequence with a relevant hook
  • Reused by aim: topic intent (Awareness); client-account activity (Expansion)
topic / post→engagers→warm list

How it runs

  1. growth_session_init — auth + workspace.
  2. growth_linkedin_search_posts — find posts on your topic.
  3. growth_linkedin_get_post_comments / growth_linkedin_get_post_reactions — pull the people engaging.
  4. growth_fetch_person_comments / growth_fetch_person_reactions — deepen the intent signal per person.
  5. Enrich, then sequence with a hook referencing what they engaged with.

Get this skill

Option A — install the GTM plugin (skill + MCP, everything)

# add the marketplace, then install the plugin
/plugin marketplace add bowtiefunnel/bowtie-funnel-Labs
/plugin install bowtie-gtm@bowtie-funnel

Bundles this skill (and the rest) plus the Growth MCP server, wired up.

Option B — just this skill

# 1) add the Growth MCP once (the engine the skill calls)
claude mcp add --transport stdio growth -- npx -y growth-mcp

# 2) copy this one skill into your Claude Code skills dir
cp -r plugins/bowtie-gtm/skills/buying-intent-miner ~/.claude/skills/

Then invoke it, or just say what you want:

/buying-intent-miner

# or, in plain language —
# "who engaged with this LinkedIn post?"
# "find people talking about GTM automation"
