llm-switchboard

local · zero-dep · MIT

A ~130-line prompt router that sits before OpenRouter. It classifies a prompt in <1ms, picks the model, and hands the ID to your OpenRouter call — cheap/free models for the workflow miles, frontier for thinking. The tier picks mirror our RevOps & Marketing agent maps.
